# Algorithms_Trends_of_Search_Recommend_Advertisement
搜广推技术演进和趋势，论文阅读，个人见解

通过搜索引擎业务的各个主要模块的算法演进，详细介绍算法的原理、问题、以及在搜广推上的应用，以及优化方向
本仓库主要结合自己在搜索引擎业务的经验，以及网络上其他人总结的经验、论文等多个渠道进行总结归纳，为大家提供一个学习分享的地方，因个人经验和能力问题，难免会存在不足的地方，所以会尽量提供参考来源

搜索引擎的主要处理模块，按系统处理用户输入的先后顺序开始，有query autocompletion或者query suggestion，query处理，召回（匹配），粗排，精排，重排，有一些搜索引擎由于会存在多种不同载体的结果，最后还会有一个混排的操作
最近快手尝试了很多端到端的方法，比如OneSug， OneSearch，OneRec等
